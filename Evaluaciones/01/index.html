HTML

HTML son las siglas de HiperText Markup Language, que puede traducirse como lenguaje de marcas o etiquetado.

El lenguaje HTML se emplea para crear paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nombre muy similar.

HTML se compone de etiquetas que se escriben entre los simbolos menos que y mayor que.

Historia de HTML

Los inicios de HTML de remontan al año 1990, cuando Tim Burners-Lee creo la primer pagina web.

Versiones de HTML

Tim Burners-Lee definio la primer version de HTML en el año 1991.

En la actualidad, la ultima version de HTML es HTML5.
